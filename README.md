# Tic-Tac-Toe

Tic-Tac-Toe playable through CLI or GUI!

How to run (CLI):
- Compile *CLIDemo.java*, *GameEngine.java*, *AI.java*
- Run
...
java CLIDemo
...

Select AI (default normal difficulty):
...
java CLIDemo AI
...

Select AI hard difficulty:
...
java CLIDemo hard
...

How to run (GUI):
...
java GUIDemo
...
or execute TicTacToe.jar
